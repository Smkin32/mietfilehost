MietFileHost
============

_Этот репозиторий часть проекта по созданию файлообменника репозиторий клиента: [тык](https://github.com/ZaroKRouK/FileStoreMIETTgBot)_

Backend сервер на java с использованием spring framework.

Архитектура и зависимости
-------------------------
* Java
* Spring Boot
* Minio s3
* REST API

Как билдить?
------------
1. Настройте переменные окружения в application.properties
2. ``` mvn clean package ```
3. ``` docker compose up ```
4. PROFIT!

Запуск тестов
-------------
Запустить тесты из папки tests
